# 题目 - 判断对称二叉树

## 题目描述

请设计一个函数判断一棵二叉树是否 轴对称 。


### 示例 1：

![](./图片/轴对称二叉树示例1.png)

* `输入`：root = `[6,7,7,8,9,9,8]`
* `输出`：`true`
* `解释`：从图中可看出树是轴对称的。

### 示例 2：

![](./图片/轴对称二叉树示例2.png)

* `输入`：root = `[1,2,2,null,3,null,3]`
* `输出`：`false`
* `解释`：从图中可看出最后一层的节点不对称。

### 节点结构

```java
public class TreeNode {
      int val;
      TreeNode left;
      TreeNode right;
      TreeNode() {}
      TreeNode(int val) { this.val = val; }
      TreeNode(int val, TreeNode left, TreeNode right) {
          this.val = val;
          this.left = left;
          this.right = right;
      }
}
```

### 提示：

0 <= `节点个数` <= 1000


## 题解

对称二叉树定义： 对于树中 任意两个对称节点 L 和 R，一定有：

* L.val = R.val ：即此两对称节点值相等。
* L.left.val = R.right.val ：即 L 的 左子节点 和 R 的 右子节点 对称；
* L.right.val = R.left.val ：即 L 的 右子节点 和 R 的 左子节点 对称。

根据以上规律，考虑从顶至底递归，判断每对左右节点是否对称，从而判断树是否为对称二叉树。

![](./图片/轴对称二叉树题解.png)


```java
class Solution {
    public boolean checkSymmetricTree(TreeNode root) {
        return checkSymmetry(root,root);
    }

    public boolean checkSymmetry(TreeNode left, TreeNode right){
        //1 - 判断左右节点是否仅有一个为null，是则返回false
        if((left == null && right != null) || (left != null && right == null)) return false;
        //2 - 判断左右节点是否均为null，是则返回true
        if(left == null && right == null) return true;
        
        //将left.left,right.right编为一组，left.right,right.left编为一组进行递归，附带上判断左右节点值是否相同
        return checkSymmetry(left.left,right.right)
                && checkSymmetry(left.right,right.left)
                && left.val == right.val;
    }
}
```
